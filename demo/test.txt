# Markdown Viewer Extension Demo

This is a **markdown file** with `.txt` extension to demonstrate the context menu preview feature.

## Features

- Right-click context menu support
- Preview any file as Markdown
- Multi-language support (28 languages)
- No file extension restriction

## Code Example

```javascript

  console.log(`Hello, ${name}!`);
```
